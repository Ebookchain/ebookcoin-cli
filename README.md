# Ebookcoin CLI

A Command line tool for [Ebookchain](http://ebookchain.org/).

## Installation:

```
npm install -g ebookcoin-cli
```

## Apps

### Add app

```
ebookcoin dapps -a
```

### Deposit

```
ebookcoin dapps -d
```

### Withdraw

```
ebookcoin dapps -w
```

## Contracts

### Add contract

```
ebookcoin contract -a
```

### Delete contract

```
ebookcoin contract -d
```

## Crypto

### Generate public key

```
ebookcoin crypto -p
```

### Generate accounts

```
ebookcoin crypto -g
```

### Help

```
ebookcoin -h
```
